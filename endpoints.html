Consulta de CFC

Endpoint
GET /api/consulta/cfc?cnpj={cnpj}
Este endpoint retorna informações sobre os Centros de Formação de Condutores cadastrados e ativos no sistema.
Documentação - Consulta CFC
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CNPJ String Não CNPJ do CFC. Se não informado, retorna todos os CFCs ativos.
Parâmetros de Saída
Parâmetro Tipo Descrição
CodigoRetorno Inteiro Código de retorno da operação (0 = sucesso, outros = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
CFCs Array Lista de CFCs com os campos: CNPJ, RazaoSocial, NomeFantasia, Email, Categoria (A/B/AB), CodigoCFC
Exemplo de Requisição
// Retorna todos os CFCs ativos GET /api/consulta/cfc // Retorna um CFC específico GET /api/consulta/cfc?cnpj=12.345.678/0001-90
Exemplo de Resposta
{ "CodigoRetorno": 0, "MensagemRetorno": "Consulta realizada com sucesso", "CFCs": [ { "CNPJ": "12.345.678/0001-90", "RazaoSocial": "Auto Escola Excelência Ltda", "NomeFantasia": "Auto Escola Excelência", "Email": "contato@autoescolaexcelencia.com.br", "Categoria": "AB", "CodigoCFC": 1234 } ] }

Consulta de Instrutor

Endpoint
GET /api/consulta/instrutor?cnpj={cnpj}&cpf={cpf}
Este endpoint retorna informações sobre os instrutores ativos de um CFC específico.
Documentação - Consulta Instrutor
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CNPJ String Sim CNPJ do CFC para filtrar os instrutores
CPF String Não CPF do instrutor. Se não informado, retorna todos os instrutores ativos do CFC.
Parâmetros de Saída
Parâmetro Tipo Descrição
CodigoRetorno Inteiro Código de retorno da operação (0 = sucesso, outros = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
Instrutores Array Lista de instrutores com os campos: CPF, Nome, Categoria, Foto (Bytes em Base64)
Exemplo de Requisição
// Retorna todos os instrutores de um CFC GET /api/consulta/instrutor?cnpj=12.345.678/0001-90 // Retorna um instrutor específico GET /api/consulta/instrutor?cnpj=12.345.678/0001-90&cpf=123.456.789-00
Exemplo de Resposta
{ "CodigoRetorno": 0, "MensagemRetorno": "Consulta realizada com sucesso", "Instrutores": [ { "CPF": "123.456.789-00", "Nome": "Carlos Silva", "Categoria": "AB", "Foto": "base64_encoded_image_data" } ] }

Consulta de Candidato

Endpoint
GET /api/consulta/candidato?cpf={cpf}&cnpj={cnpj}
Este endpoint retorna os dados da matrícula de um candidato em um CFC específico.
Documentação - Consulta Candidato
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CPF String Sim CPF do candidato que se deseja consultar
CNPJ String Sim CNPJ do CFC onde o candidato está matriculado
Parâmetros de Saída
Parâmetro Tipo Descrição
CodigoRetorno Inteiro Código de retorno da operação (0 = sucesso, outros = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
Candidatos Array Dados do candidato com campos: CPF, Renach, Nome, DataAbertura, Curso (Teorico, PraticoB, C, D, E), EmissaoLADV, VencimentoLADV, Foto, Digitais
Exemplo de Requisição
// Consulta os dados de um candidato GET /api/consulta/candidato?cpf=123.456.789-00&cnpj=12.345.678/0001-90
Exemplo de Resposta
{ "CodigoRetorno": 0, "MensagemRetorno": "Consulta realizada com sucesso", "Candidatos": [ { "CPF": "123.456.789-00", "Renach": "123456789", "Nome": "João da Silva", "DataAbertura": "2023-01-15", "Curso": { "Teorico": true, "PraticoB": true, "C": false, "D": false, "E": false }, "EmissaoLADV": "2023-05-20T10:30:00", "VencimentoLADV": "2023-11-20T10:30:00", "Foto": "base64_encoded_image_data", "Digitais": ["base64_encoded_image_data1", "base64_encoded_image_data2"] } ] }

Consulta de Agendamento Prático

Endpoint
GET /api/consulta/agendamento-pratico?cnpj={cnpj}&data={data}&placa={placa}
Este endpoint retorna os agendamentos de aulas práticas para um CFC em uma data específica.
Documentação - Consulta Agendamento Prático
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CNPJ String Sim CNPJ do CFC para filtrar os agendamentos
Data Date Sim Data para a qual se deseja consultar os agendamentos
Placa String Não Placa do veículo. Se informado, retorna apenas os agendamentos para este veículo.
Parâmetros de Saída
Parâmetro Tipo Descrição
CodigoRetorno Inteiro Código de retorno da operação (0 = sucesso, outros = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
AgendamentosPratico Array Lista de agendamentos com campos: CPF, Renach, Placa, CPFInstutor, HoraInicio DateTime, HoraFim DateTime, CodigoAula
Exemplo de Requisição
// Retorna todos os agendamentos de uma data GET /api/consulta/agendamento-pratico?cnpj=12.345.678/0001-90&data=2023-05-20 // Retorna agendamentos para um veículo específico GET /api/consulta/agendamento-pratico?cnpj=12.345.678/0001-90&data=2023-05-20&placa=ABC1D23
Exemplo de Resposta
{ "CodigoRetorno": 0, "MensagemRetorno": "Consulta realizada com sucesso", "AgendamentosPratico": [ { "CPF": "123.456.789-00", "Renach": "123456789", "Placa": "ABC1D23", "CPFInstutor": "987.654.321-00", "HoraInicio": "2023-05-20T08:00:00", "HoraFim": "2023-05-20T09:00:00", "CodigoAula": 1001 }, { "CPF": "111.222.333-44", "Renach": "987654321", "Placa": "XYZ9W87", "CPFInstutor": "987.654.321-00", "HoraInicio": "2023-05-20T09:00:00", "HoraFim": "2023-05-20T10:00:00", "CodigoAula": 1002 } ] }

Confirmação de Aula Prática

Endpoint
POST /api/confirmacao/aula-pratica
Este endpoint envia a confirmação de uma aula prática monitorada, incluindo as coletas biométricas de aluno e instrutor.

Coletas Biométricas

Entrada do Instrutor
Imagem da digital/foto
Saída do Instrutor
Imagem da digital/foto
Entrada do Aluno
Imagem da digital/foto
Saída do Aluno
Imagem da digital/foto
Documentação - Confirmação Aula Prática
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CodigoAula Int Sim Código único identificador da aula
CPFAluno String Sim CPF do aluno que participou da aula
CPFInstrutor String Sim CPF do instrutor que ministrou a aula
CNPJCFC String Sim CNPJ do CFC onde a aula foi realizada
Placa String Sim Placa do veículo utilizado na aula
EntradaInstrutor Object Sim Dados de coleta biométrica do instrutor no início da aula
SaidaInstrutor Object Sim Dados de coleta biométrica do instrutor no fim da aula
EntradaAluno Object Sim Dados de coleta biométrica do aluno no início da aula
SaidaAluno Object Sim Dados de coleta biométrica do aluno no fim da aula
Parâmetros de Saída
Parâmetro Tipo Descrição
DigitalAlunoInicio Enum Status da digital do aluno no início (Invalido/NaoExisteBiometria)
DigitalAlunoFim Enum Status da digital do aluno no fim (Invalido/NaoExisteBiometria)
DigitalInstrutorInicio Enum Status da digital do instrutor no início (Invalido/NaoExisteBiometria)
DigitalInstrutorFim Enum Status da digital do instrutor no fim (Invalido/NaoExisteBiometria)
CodigoRetorno Int Código de retorno da operação (0 = sucesso, 1 = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
Exemplo de Requisição
POST /api/confirmacao/aula-pratica { "CodigoAula": 1001, "CPFAluno": "123.456.789-00", "CPFInstrutor": "987.654.321-00", "CNPJCFC": "12.345.678/0001-90", "Placa": "ABC1D23", "EntradaInstrutor": { "Coleta": { "Hora": "2023-05-20T08:00:00", "Tipo": "Digital", "Digital": "base64_encoded_image_data", "Foto": null } }, "SaidaInstrutor": { "Coleta": { "Hora": "2023-05-20T09:00:00", "Tipo": "Digital", "Digital": "base64_encoded_image_data", "Foto": null } }, "EntradaAluno": { "Coleta": { "Hora": "2023-05-20T08:00:00", "Tipo": "Foto", "Digital": null, "Foto": "base64_encoded_image_data" } }, "SaidaAluno": { "Coleta": { "Hora": "2023-05-20T09:00:00", "Tipo": "Foto", "Digital": null, "Foto": "base64_encoded_image_data" } } }
Exemplo de Resposta
{ "DigitalAlunoInicio": "NaoExisteBiometria", "DigitalAlunoFim": "NaoExisteBiometria", "DigitalInstrutorInicio": "Valido", "DigitalInstrutorFim": "Valido", "CodigoRetorno": 0, "MensagemRetorno": "Aula confirmada com sucesso" }
Observações

A aula não será aceita se a digital não for validada e não houver foto no envio.

Confirmação de Aula Teórica

Endpoint
POST /api/confirmacao/aula-teorica
Este endpoint envia a confirmação de uma aula teórica monitorada.
Documentação - Confirmação Aula Teórica
Parâmetros de Entrada
Parâmetro Tipo Obrigatório Descrição
CodigoTurma Int Sim Código único identificador da turma
CodigoSala Int Sim Código da sala onde a aula foi ministrada
CPFInstrutor String Sim CPF do instrutor que ministrou a aula
CNPJCFC String Sim CNPJ do CFC onde a aula foi realizada
QuantidadeAlunosMonitorado int Sim Quantidade de alunos monitorados presentes na aula
HoraInicio DateTime Sim Data e hora de início da aula
HoraFim DateTime Sim Data e hora de término da aula
Parâmetros de Saída
Parâmetro Tipo Descrição
CodigoRetorno Int Código de retorno da operação (0 = sucesso, 1 = erro)
MensagemRetorno String Mensagem descritiva sobre o resultado da operação
Exemplo de Requisição
POST /api/confirmacao/aula-teorica { "CodigoTurma": 2001, "CodigoSala": 1, "CPFInstrutor": "987.654.321-00", "CNPJCFC": "12.345.678/0001-90", "QuantidadeAlunosMonitorado": 25, "HoraInicio": "2023-05-20T08:00:00", "HoraFim": "2023-05-20T10:00:00" }
Exemplo de Resposta
{ "CodigoRetorno": 0, "MensagemRetorno": "Aula teórica confirmada com sucesso" }
