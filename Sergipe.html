Documentação Técnica - Detran-SE

Este pacote de integração atua como um wrapper para simplificar a comunicação com os endpoints oficiais do Detran Sergipe, encapsulando o fluxo de autenticação e as requisições HTTP. O gerenciamento de autenticação é feito pela classe **TokenSE**, que é responsável por obter e gerenciar o token JWT (JSON Web Token).

Consulta de Veículos por CFC

Visão Geral da Consulta
Este é um endpoint crucial para a gestão de frotas de um Centro de Formação de Condutores (CFC). Ele permite que você retorne todos os veículos associados a um CFC específico, com a opção de filtrar a busca por uma placa para obter informações detalhadas, como chassi, categoria, tipo, cor, ano de fabricação e situação do licenciamento.
Documentação Detalhada - Veículos
Endpoint
GET /Monitoramento/veiculo/{codigoCfc}
Parâmetros de Entrada na URL
Parâmetro Tipo Obrigatório Descrição
codigoCfc string Sim Código do CFC no Detran-SE.
placa string Não Filtro por placa específica.

Consulta de Centros de Formação (CFCs)

Visão Geral da Consulta
Este endpoint é a porta de entrada para buscar informações cadastrais e de credenciamento dos CFCs. A API permite listar todos os CFCs ou filtrar por um código específico. As informações retornadas incluem detalhes de contato (endereço, telefone, e-mail), a situação do credenciamento e os dados dos diretores, que são essenciais para a verificação de credenciais e a conformidade regulatória.

Endpoint
GET /Monitoramento/Cfc
Parâmetros de Entrada na URL

Consulta de Instrutores por CFC

Visão Geral da Consulta
Este endpoint é utilizado para validar e consultar os instrutores credenciados em um CFC. É possível listar todos os instrutores ativos de uma unidade ou buscar um instrutor específico por seu registro profissional. A resposta retorna dados profissionais, como situação de credenciamento, categorias habilitadas e datas de vencimento, além dos templates biométricos (digitais e foto) que são essenciais para a autenticação durante o envio de aulas.
Documentação Detalhada - Instrutores
Endpoint
GET /Monitoramento/Instrutor/{codigoCfc}
Parâmetros de Entrada na URL
Parâmetro Tipo Obrigatório Descrição
codigoCfc string Sim Código do CFC no Detran-SE.
registroInstrutor string Não Filtro por registro de instrutor específico.

Consulta de Candidato por CPF

Visão Geral da Consulta
Este endpoint permite a verificação do processo de habilitação de um candidato junto ao Detran-SE. Ao fornecer o CPF, tipo de aula, categoria e código do CFC, é possível obter os dados do processo, como o número do Renach, o nome completo e as datas de validade do processo. A resposta também pode incluir a biometria facial e digital do candidato, que são cruciais para confirmar sua identidade antes e depois das aulas.
Documentação Detalhada - Candidato
Endpoint
GET /Monitoramento/Cpf/{cpf}/{tipoAula}/{categoria}/{codigoCfc}
Parâmetros de Entrada na URL
Parâmetro Tipo Obrigatório Descrição Valores Válidos
cpf string Sim CPF do candidato (somente números). 11 dígitos
tipoAula string Sim Tipo de aula. "T" (Teórica), "P" (Prática)
categoria string Sim Categoria desejada. "A", "B", "AB", "ACC"
codigoCfc string Sim Código do CFC no Detran-SE. Alfanumérico

Envio de Aulas (Teórica e Prática)

Visão Geral do Envio de Aulas
Este é um dos endpoints mais importantes da API, usado para registrar oficialmente uma aula (teórica ou prática) no sistema do Detran-SE. A requisição `POST` deve conter um corpo (body) com todos os detalhes da aula, incluindo CPF do aluno, tipo de aula, código do CFC, registro do instrutor, datas, horários e, para aulas práticas, a placa do veículo. É crucial garantir que os dados sejam enviados no formato correto, pois este é o passo final para a validação da carga horária do aluno. A API retorna o status da operação para confirmar se a aula foi cadastrada com sucesso ou se houve algum erro.
Documentação Detalhada - Envio de Aulas
Endpoint
POST /Monitoramento/aula/cadastrar
Estrutura do Body (AulaEnvioModel)
{
  "cpf": "12345678901",
  "tipoAula": "P",
  "categoria": "B",
  "codigoCfc": "CFC54321",
  "registroInstrutor": 12345,
  "dataAula": 20230815,
  "horaInicio": 830,
  "horaFim": 930,
  "codigoMateria": "L",
  "placa": "ABC1D23"
}
Observações Importantes
  • O campo **categoria** é obrigatório apenas para aulas práticas ("P").
  • O campo **codigoMateria** é obrigatório para aulas teóricas ("T", "A", "R").
  • O campo **placa** é obrigatório para aulas práticas ("P").
  • Campos que não são obrigatórios devem ser enviados com uma string vazia se não forem aplicáveis.
